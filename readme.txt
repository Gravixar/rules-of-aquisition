== Description ==

This is not just a plugin, it symbolizes the hope and enthusiasm of an entire generation summed up in in many rules written most famously by Gint, the first Grand Nagus of the Ferengi people. When activated you will randomly see a rule from <cite>Ferengi Rules of Acquisition</cite> in the upper right of your admin screen on every page.
